Employee Transfer

This is employee transfer view screen. We can add, edit, and delete the employee transfer details value
To add a new employee transfer details by click the add button
To edit employee transfer details by click the edit button
To delete the employee transfer details by click the delete button
To filter employee transfer by using alphabetic filter and search by name
 
 

Employee transfer add page
 
To add a employee transfer by click the add button in employee transfer view screen
Click the Add button in employee transfer view screen
Select company, branch, designation, department, office
Select category, reporting to, staff type, transfer order issue date
Select date of releaving, authorized by
Enter reason.
The * mark symbol are mandatory
The * mark field validate when click the submit button
Click the transfer button
The employee transfer successfully added
 
 

Employee transfer edit page
 
Edit or change a employee transfer
Choose or tick the check box for employee transfer in view screen
Click the Add button in employee transfer view screen

Change company, branch, designation, department, office.
Change category, reporting to, staff type, transfer order issue date

Change date of relieving, authorized by.
Modify reason.
The * mark symbol are mandatory
The * mark field validate when click the submit button
Submit the update button
The employee transfer successfully updated
