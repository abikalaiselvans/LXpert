Loan Credit

This Loan credit view screen. We can add, edit, and delete the Loan credit
To add a new Loan credit by click the add button
To edit Loan credit by click the edit button
To delete the Loan credit by click the delete button
To filter Loan credit by using alphabetic filter ,office and loan type.
 
 

Loan credit add page
 
To add a loan credit by click the add button in loan credit view screen
Click the Add button in loan credit view screen
Select employee name, loan type, credit date
Enter loan amount and description
Select authorized by.
The * mark symbol are mandatory
The * mark field validate when click the submit button
Click the add button
The loan credit successfully added
 
 

Loan credit edit page
 
Edit or change a loan credit
Choose or tick the check box for loan credit in view screen
Click the edit button in loan credit view screen
Change loan type, credit date.
Enter loan amount and description
Select authorized by
The * mark symbol are mandatory
The * mark field validate when click the edit button.
Click the edit button
The loan credit successfully updated
