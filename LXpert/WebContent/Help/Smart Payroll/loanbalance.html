Loan Balance

This Loan balance view screen. We can add, edit, and delete the Loan balance

To display the balance amount calculated by credit and debit
To add credit amount by using credit button
To add debit amount by using debit button.
To filter Loan balance by using alphabetic filter ,office and loan type.
 
 

Loan balance credit page
 
To add a loan credit by click the credit button in loan balance view screen
Select employee name, loan type, credit date
Enter loan amount and description
Select authorized by

The * mark symbol are mandatory.
The * mark field validate when click the submit button
Click the add button
The loan credit successfully added.
 
 

Loan balance debit page
 
To add a loan debit by click the debit button in loan balance view screen
Select employee name, loan type, debit date
Loan balance amount is appear on page according an employee id
Loan balance amount validate with debit amount .we can’t enter debit
amount greater than loan balance amount
Enter loan amount and description
Select cash mode
The * mark symbol are mandatory
The * mark field validate when click the add button
Click the add button
The loan debit successfully added
