Loan Debit

This Loan debit view screen. We can add, edit, and delete the Loan debit.
To add a new Loan debit by click the add button
To edit Loan debit by click the edit button
To delete the Loan debit by click the delete button
To filter Loan debit by using alphabetic filter ,office and loan type.
 
 

Loan debit add page
 
To add a loan debit by click the add button in loan debit view screen
Click the Add button in loan debit view screen
Select employee name, loan type, debit date
Loan balance amount is appearing on page according a employee id
Loan balance amount validate with debit amount .we can’t enter debit
amount greater than loan balance amount

Enter loan amount and description.
Select cash mode.
The * mark symbol are mandatory
The * mark field validate when click the add button
Click the add button
The loan debit successfully added
 
 

Loan debit edit page
Edit or change a loan debit
Choose or tick the check box for loan debit in view screen
Click the edit button in loan debit view screen
Change loan type, debit date
Enter loan amount and description
Select cash mode
The * mark symbol are mandatory
The * mark field validate when click the edit button
Click the edit button
The loan debit successfully updated
