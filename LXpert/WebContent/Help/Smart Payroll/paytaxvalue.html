Paytax Value
This is pay tax value view screen. We can add, edit the pay tax value.
To add a new pay tax value by click the add button.
To edit pay tax value by click the edit button
To delete the pay tax value by click the delete button
To filter pay tax value by using alphabetic filter and tax name.
 
 

Pay tax value add page
 
To add a pay tax value by click the add button in pay tax value view screen
Click the Add button in pay tax value view screen
Select the tax name
Enter the greater than, less than,percentage,amount,description

The * mark symbol are mandatory.
The * mark field validate when click the submit button
Click the submit button
The pay tax value successfully added
 
 

Pay tax value edit
 
Edit or change a pay tax value.
Choose or tick the check box for pay tax value in view screen

Click the Edit button in pay tax value view screen.
Change the tax name
Modify the greater than, less than, percentage, amount, and description.
The * mark symbol are mandatory
The * mark field validate when click the submit button.
Click the update button.
The pay tax value successfully updated
