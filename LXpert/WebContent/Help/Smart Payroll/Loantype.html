Loan Type
This is loan type view screen. We can add, edit the loan type
To add a new loan type by click the add button.
To edit loan type by click the edit button

To delete the loan type by click the delete button.
To filter loan type by using alphabetic filter.
 
 

Loan type add page :
 
To add a loan type by click the add button in loan type view screen
Click the Add button in loan type view screen
Enter the name and description
The * mark symbol are mandatory
The * mark field validate when click the submit button.
Submit the add button
The loan type successfully added.
 
 

Loan type edit page :
 
Edit or change a Loan type.
Choose or tick the check box for Loan type in view screen
Click the Edit button in Loan type view screen
Modify the name and description
The * mark symbol are mandatory
The * mark field validate when click the submit button.
Submit the update button
The Loan type successfully updated
