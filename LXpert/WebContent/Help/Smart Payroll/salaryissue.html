Salary Issue
We can classified salary issue by category wise and employee id wise
To classified salary issue by click category wise link
To classified salary issue by click employee id wise
 
 

Category wise
 

Category wise salary issue page is appear by click category wise link.
Select office,department,category,month,year
Select issue date,issue mode,issue bank name
Enter the description
The * mark symbol are mandatory
The * mark field validate when click the submit button

Click issue button.

The salary credited successfully

 
 
 

Employee id wise
 

Employee id wise salary issue page is appear by click Employee id wise link.

Enter the employee id.
Select month, year,issue date,issue mode,issue bank name
Enter the description

The * mark symbol are mandatory.
The * mark field validate when click the submit button
Click issue button

The salary credited successfully
