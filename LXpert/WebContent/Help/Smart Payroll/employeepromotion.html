Employee Promotion

This is employee promotion view screen. We can add, edit, and delete the employee promotion details
To add a new employee promotion details by click the add button
To edit employee promotion details by click the edit button
To delete the employee promotion details by click the delete button
To filter employee promotion by using alphabetic filter and search by name
 
 

Employee promotion add page :
 
Edit or change a employee promotion
Choose or tick the check box for employee promotion in view screen
Click the Add button in employee promotion view screen
Select promotion date, transfer designation
Enter reason
Select change salary details
The * mark symbol are mandatory
The * mark field validate when click the submit button
Click the submit button
The employee promotion successfully added.

Enter reason.
