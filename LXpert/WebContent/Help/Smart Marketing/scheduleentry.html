Schedule Entry
This is schedule entry view screen. We can add, edit, and delete the schedule entry
To add a new schedule entry by click the add button
To edit a schedule entry by click the edit the edit button
To delete a Schedule entry by click the delete button
To filter schedule entry by alphabetically.
To filter schedule entry by using day,month,year.
 
 

Schedule Entry add page
You want to add a schedule entry by click the add button in schedule entry  view screen
Enter the  customer name, title
Select the starting date and ending date
Tick the all day check box
The * mark fields are mandatory.
The * mark fields validate when click the submit button
Click the submit button
The schedule entry added successfully.
 
 

Schedule Entry edit page
You want to edit a schedule entry by click the edit button in schedule entry  view screen
Modify  the  customer name, title
Change  the starting date and ending date
Tick the all day check box
The * mark fields are mandatory
The * mark fields validate when click the submit button.
Click the submit button
The schedule entry update successfully.
