Designation
This is Designation view screen. We can add, edit, and delete the Designation
To add a new Designation by click the add button
To edit Designation by click the edit button
To delete the Designation by click the delete button
To filter the designation by using alphabetically
 
 

Add page :
 
To add a Designation by click the add button in Designation view screen
Enter the Name, description
Select the Active
Click the add button
The * mark fields are mandatory
The * mark fields validate during the submission
