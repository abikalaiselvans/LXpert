Customer Type
This is customer type view screen. We can add, edit and delete the customer type
To add a new customer type by click the add button
To edit customer type by click the edit button
To delete customer type by click the delete button
To filter customer type by using alphabetic filter
 
 

Add page :
 
To add a customer type by click the add button in customer type view screen
Click the Add button in customer type view screen
Enter customer type name
The * mark symbol are mandatory
The * mark field validate when click the submit button
click the add  button
The customer type successfully added
 
 

Edit page :
 
Edit or change customer type
Choose or tick the check box for customer type in view screen
Click the Edit button in customer type view screen
Modify the customer type name
The * mark symbol are mandatory
The * mark field validate when click the submit button
Click the update button
The customer type successfully updated.
