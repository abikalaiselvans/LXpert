Item

This is item view screen. We can add, edit and delete the item
To add a new item by click the add button
To edit item  by click the edit button
To delete the item  by click the delete button
To filter item by using alphabetic filter,item group,search item name.
 
 

Add page :
 
To add a item by click the add button in item view screen
Click the Add button in item  view screen
Enter the item name,purchase price,sales price
Select active,item group,model,color,division and tax
Enter item description,purchase description,sales description
Select warranty info

The * mark symbol are mandatory.
The * mark field validate when click the submit button
click the add button
The item successfully added
 
 

Edit page :
 
Edit or change item
Choose or tick the check box for item in view screen

Click the Edit button in item view screen.
Modify item name, purchase price, sales price
Change active, item group, model, color, division and tax
Modify item description, purchase description and sales description
Change warranty info
The * mark symbol are mandatory
The * mark field validate when click the submit button
Click the update button
The item successfully updated
