EMD Approval
This is emd approval view screen.we can received ,accept the emd.
To received  the emd by using received  link.
To accept emd  by using accept link.
To filter emd approval  by using alphabetic filter, office, emd type, month, year ,accept and returned.
 
 

Accept page :
 
To accept the emd by using accept link in emd approval view page
Select bank,approval.
Enter dd/cheque number,description
Click the approval button.
Emd successfully accepted.
 
 

Received page :
 
To received the emd by using received  link in emd approval view page.
Choose returned.
Enter date of dd,description
Click the received button
Emd successfully received.
