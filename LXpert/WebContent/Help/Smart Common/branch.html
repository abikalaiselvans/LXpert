Branch
This is Branch view screen. We can add, edit, and delete the Branch
To add a new Branch by using the add button
To edit Branch by  using the edit button
To delete the Branch by using the delete button
To filter the Branch by alphabetically and company.
 
 

Branch add
To add a new branch by click the add button in branch view screen
Select company name.
Enter the branch name, legal name, street.
Select the country,state,district,city
Enter the postal, area code, phone ,fax, email
Enter the website, register number,tngst number,cst number, tin number.
Enter local tax number, pan number
Choose live status.
The * mark fields are mandatory
The * mark fields are validation when submit this page
click the add button.
The branch successfully added
 
 

Branch edit
 
Edit or change a branch.

Change the company name.
Modify the branch name, legal name, street
Change the country, state, district, and city.
Modify the postal, area code, phone, fax, email
Modify the website, register number, tngst number, cst number, tin number.
Modify  local tax number, pan number
