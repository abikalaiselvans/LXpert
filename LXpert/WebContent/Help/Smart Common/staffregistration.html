Staff Registration
This is Staff Registration view screen. We can add & edit  the Staff Registration
To add a new Staff Registration by click the add button
To edit Staff Registration by click the edit button
To filtered the staff information by  using  the Company, branch, department, designation
The easiest way is search name or id wise
The temporarily Id can also possible in staff registration screen
 
 

Staff registration add :
 
To add a new Staff Registration by click the add button in Staff Registration view screen
The * mark field is mandatory
If the employee come through the HRM module then select the HRM ID.
After selecting the HRM ID the employee information’s are automatically filled
Otherwise information’s are entering through manually
Enter the staff name
Enter the father’s name. if married then enter the spouse name
Select gender blood group,marital status and date of joining
Enter the wedding date if married
Enter official mail if you can provide
Enter the personal mail ids and mobile number for communication
Select the highest qualification, reporting to
Select dispensary and category
Select the company and branch
Select  state, district, office, department, designation, staff type
Enter a employee gross. Then select the salary split up
The salary split up is automatically. [ After submitting the button, the gross divided into basic and allowances its depends on salary slab in smart payroll ] .
The salary split up is manual. [ After submitting the button, the screen will appear that asking the basic and allowances. The split up will enter manually.]
The salary type is consolidate or labor the gross  is put into basic. There is no split up
Select the bank name and account number. There is no bank kindly select the no bank and account number is empty / DD/ cheque
And also enter the IFSC code of that bank. Because the salary is net transfer
Please provide the information about leave carry, finger print login, Management
Provide pay slip hide
If the employee is a head then tick the reporting head
Provide conveyance clear option.
Select the attendance cycle
The attendance cycle is divide into 2 type such as 1-31 called as cycle-1, 20th to 19th called as cycle-2.
Select salary type. The salary type is having 3 types
CTC - Salary calculated in (month salary*12/365) *working days - including Sundays
Month - Salary calculated in (month salary/No of days in a Month) *working days - including Sundays
Working - Salary calculated in (month salary/(No of days in a Month -
No of sundays in a month)) *working days
Give the permanent Address and communication address
You know the pin code, that number type to pin code field it gives the list then select the correct one its automatically select the country, state, district, city and phone numbers
The communications is same by click the Copy Permanent to Communication Address
Please provide the PF & ESI details
The PF is YES the PF Type is  Amount and give the PF Number
The PF is COMMON the PF Type is  Percentage and give the PF Number
The PF is NO the PF Type is  no and PF Number also no
The ESI is YES the PF Type is  Amount and give the ESI Number
The ESI is COMMON the PF Type is  Percentage and give the ESI Number.
The PF is ESI the PF Type is  no and ESI Number also no
Provide the PF date of joining. The date is equal or greater than the date of joining
You have addition information of PAN, PASS PORT, DRIVING License and previous details. Please enter it.
The above details are entered the submit the submit button
During submission the validation occurs. If the validation is true then the employee id created and asking one more question to you. You want to add one more staff registration yes or no?
You click the yes button the new staff registration screen comes
You click the no button the staff registration view screen comes
Click the add button
Staff information successfully added
