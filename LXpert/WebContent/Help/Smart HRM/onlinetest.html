Online Test
The  test applicant only loaded to the screen in dropdown box
Select the applicant and submit the button
 
 

Online test Start
 
After the submission. The screen comes like this
The default test name and no of question displayed. This is depends on job design.
we change the attempt no of question field.
Submit the button.
 
 

Instruction page
 
This is the message screen for online test
The online test can start by click the start test button
 

Test question and answer page
 
The screen displayed the question and answer
Please choose the correct answer
In the top screen, the clock runs
All the questions are objective types
After attempting the question by click the submit  button
If time is end before you finish the question when the page automatically submit
 
 
 

Test result page
 
This is the validation screen
Each and every question has validated.
If the question is correct then tick image displayed.
If the question is wrong then cross image displayed.
Then finally the result is displayed as pass or fail
The print out options is available in Test result page
 
 

Result view page
 
This is result view screen. We can add, edit, and delete the result
Make to add a new result by click the add button.which similar
process to above test page.
Make to edit result by click the edit button
Want to delete the result by click the delete button.
 
 

Result edit page
 
This is the validation screen for result
Each and every question has validated.
If the question is correct then tick image displayed.
If the question is wrong then cross image displayed
Then finally the result is displayed as pass or fail
The print out options is available in Test result page
We can’t edit data in this page.
