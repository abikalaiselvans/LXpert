First interview Test View Page
This is First interview view screen. We can add, edit, and delete the First interview
Make to add a new First interview by click the add button
Make to edit First interview by click the edit button
Want to delete the First interview by click the delete button
 
 

First interview Add:
You want to add a new First interview entry by click the add button in First interview view screen
Select the applicant name
Select the position and qualifications
Select vehicle status and relocation yes or no
Enter the prepared location
Enter the date of interview, total experience, age, current salary and expected salary
Select notice period
Enter field work, referred by
Select the skill information’s
Select interviewed by
Submit the add button
Enter the comments and description.
The * mark fields are mandatory.
The * mark fields validate during the submission
The First interview successfully added
 
 

First interview Edit:
Edit or change a First interview
Choose or tick the check box for First interview record details in view screen
Click the Edit button in First interview view screen.
Select the applicant name
Select the position and qualifications
Select vehicle status and relocation yes or no
Change  the prepared location
Change the date of interview, total experience, age, current salary and expected salary
Select notice period
Change field work, referred by
Select the skill information’s
Select interviewed by
Enter the comments and description.
The * mark fields are mandatory
The * mark fields validate during submit
Click the submit button.
The First interview successfully updated
