Mobile Contacts
This is mobile contact view screen. We can add, edit, and delete the mobile contact
To add a new mobile contact by click the add button
To edit mobile contact by click the edit button
To delete mobile contact by click the delete button
The view mobile contact record filter by alphabetically
To filter mobile contact record by using group and search by Name/id
 
 

Mobile Contact add page
You want to add a mobile contact by click the add button in mobile contact  view screen
Enter the  name, mobile,Email,city.
Select the group and age.
Click the submit button.
The * mark fields are mandatory
The * mark fields validate when click the submit button
The mobile contact added successfully
 
 

Mobile Contact edit page
You want to add a mobile contact by click the add button in mobile contact  view screen
Modify  the  name, mobile,Email,city
Change the group and age.
Click the submit button.
The * mark fields are mandatory
The * mark fields validate when click the submit button
The mobile contact update successfully
