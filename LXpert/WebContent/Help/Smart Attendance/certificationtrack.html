Certification Track
This is Certification Track view screen. We can add, edit, and delete the Certification Track.
To add a new Certification by click the add button
To edit Certification by click the edit button
To delete the Certification by click the delete button
 
 

Certification Track Add

To add a new Certification entry by click the add button in Certification view screen.
Select the employee name
Select the completed Certification
Submit the add button.
The Certification successfully added.
 
 

Certification Track Edit

 
Edit or change a Certification
Choose or tick the check box for Certification  Track  in view screen
Click the Edit button in Certification view screen
Select the Certification
Click the Submit button.
The Certification successfully updated
