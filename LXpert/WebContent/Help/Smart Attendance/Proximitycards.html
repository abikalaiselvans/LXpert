Proximity Cards
This is proximity card view screen. We can add, edit, and delete the proximity card
Make to add a new proximity card for click the add button.
Make to edit proximity card for click the edit button
Want to delete the proximity card for click the delete button.
 
 

Proximity card Add:
You want to add a new proximity card by click the add button in proximity card view screen.
Select the staff name
Enter the proximity card number
Submit the add button
The proximity card successfully added
 
 
Proximity card Edit:
 
Edit or change a proximity card,
Choose or tick the check box for proximity card view screen
Click the Edit button in proximity card view screen
Change the proximity card number.
Submit the button.
The proximity card successfully updated.
