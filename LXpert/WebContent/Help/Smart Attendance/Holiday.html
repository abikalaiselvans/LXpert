Holiday
This is Holidays view screen. We can add, edit, and delete the Holidays
To add a new Holidays by click the add button
To edit Holidays by click the edit button
To delete the Holidays by click the delete button
To filtered by alphabetically month, year, date, office and staff
 
 

Holidays Add
 
To add a new Holiday entry by  click the add button in Holiday view screen.
Select the employee
Enter the from-date and to-date
Enter the holiday name
The * mark symbol are mandatory
The * mark field validate when click the submit button
Submit the add button
The Holiday successfully added
 
 

Holidays Edit
 
Edit or change a Holiday
Choose or tick the check box for Holiday view screen.
Click the Edit button in Holiday view screen
Change the date
Change the description.
The * mark symbol are mandatory.
The * mark field validate when click the submit button.
Submit the button.
The Holiday successfully updated.
