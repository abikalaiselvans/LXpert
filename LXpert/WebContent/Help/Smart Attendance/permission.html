Permission
This is  Permission view screen. We can add,edit,delete the Permission.
To add a new Permission by click the add button
To edit a Permission by click the edit button.
To delete a Permission by click delete button
To filtered Permission by alphabetically,month,year,date,staff,office
 
 

Permission Add
 
To add a new permission by click the add button in permission view screen.
Choose the staffid.
Select the date.
Choose the day.
Select from time & to time.
Enter the reason.
The * mark symbol are mandatory
The * mark field validate when click the submit button
Click the submit button
The permission successfully added
 
 

Permission Edit

Click the Edit button in permission view screen
Change the date day type.
Select from time and to time
Enter the reason.
The * mark symbol are mandatory
The * mark field validate when click the submit button
Click the update button
